AI Hybrid Network Intrusion Detection System (NIDS)

A full AI-driven Hybrid NIDS integrating supervised machine learning (Random Forest, HGB, MLP) with unsupervised deep learning (Autoencoder) for anomaly detection. Trained and tested on real-world datasets — TII-SSRC-23 (27.5GB) and CIC-IDS-2018 — achieving strong generalization across environments. Includes a SOC-style web dashboard where users can upload CSV or PCAP traffic files, view predictions, visualize detected attacks, and provide feedback for model retraining.

  • Hybrid detection logic: binary ensemble + Autoencoder for zero-day anomaly detection
  • Admin-controlled feedback-driven retraining workflow
  • Web dashboard built with Node.js, Express, MongoDB, Chart.js, and Bootstrap 5
  • ML pipeline in Python using TensorFlow/Keras, Scikit-learn, and CICFlowMeter
